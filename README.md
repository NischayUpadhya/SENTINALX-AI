# 📌 Overview

SentinelX AI is a next-generation AI-powered crypto monitoring and risk intelligence platform designed to analyze cryptocurrency markets, identify suspicious activity, monitor risk exposure, and provide autonomous trading insights in real time.

The platform combines:
- AI-based market analysis
- Scam detection
- Sentiment monitoring
- Trading recommendations
- Risk evaluation
- Real-time dashboard visualization

# ⚙️ Installation Guide

## Clone Repository

```bash
git clone https://github.com/NischayUpadhya/SENTINALX-AI.git
```

---

## Frontend Setup

```bash
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:3000
```

---

## Backend Setup

```bash
cd Backend
pip install -r requirements.txt
uvicorn main:app --reload
```

Backend runs on:

```bash
http://127.0.0.1:8000
```
